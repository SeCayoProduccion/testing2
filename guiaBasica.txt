¿Cómo configurar mi usuario de manera local dentro de mi repositorio?
Tienes que hacer esto en cada repositorio nuevo..

git config user.name "xxx"
git config user.main "x@xx"

Para comprobar que esté hecho de manera local es esto..
git config user.name
git config user.mail
Te deben regresar el usuario y mail que están configurados, si no regresa algo significa que no hay ninguno establecido

¿Cómo clonar un proyecto?
Estando en git..
$ git clone urlDelProytecto


¿Cómo modificar un archivo? 
Estando en el proyecto..
$ nano nombreDelArchivo
ctrl x
yes
enter


¿Cómo subir un archivo/hacer push?
$ git add nombreDelArchivo
$ git commit -m 'Descripción'
$ git push


¿Cómo hacer un pull?
$ git pull

Si te aparece el error de divergent branchs hay que hacer esto..
$ git pull --no-rebase

¿Cómo cambiar el usuario que está configurado en la pc?
git config user.name 'Tu nombre'
git config user.email 'xxxx@xxx.xxx'

¿Cómo cambiar el nombre a un archivo?
$ mv nombreArhivo nombreArchivo.txt
